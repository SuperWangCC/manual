精简修改器–knight翻译¶

精简修改器允许你在尽量不改变网面形状的情况下，减少其顶点、面的数量。

This is not usually used on meshes which have been created by modeling carefully and economically (where all vertices and faces are necessary to correctly define the shape). But if the mesh is the result of complex modeling, sculpting and/or applied Subdivision Surface/Multires modifiers, the Decimate modifier can be used to reduce the polygon count for a performance increase, or simply remove unnecessary vertices and edges.

精简修改器是一种快速和简单的以非破坏性方式减少网面多边形数量的方式。该修改器论证了网面修改器系统的优势，因为它展示了一种交互式地并安全地使用修改器的操作，它是标准永久的，且会破坏原始网面数据。

不同于大多数现有的修改器，精简修改器不允许你在编辑模式下查看所作的更改。

选项¶

decimate modifier.

精简类型
塌陷

渐进式地同时合并顶点，会考虑网面的形状。

比率

精简后保留面数量的比例。

  • On 1.0: the mesh is unchanged.
  • On 0.5: edges have been collapsed such that half the number of faces remain (See Note below).
  • On 0.0: all faces have been removed.

注解

虽然 比率 与剩余面的数量直接成正比，但在计算比率时，三角形也会使用。

这意味着，如果你的网面包含四边形，剩余面的数量会大于预期数量，因为如果他们的边缘不塌陷的话，四边形是保持不变的。

在禁用了 三角化 选项，上面的话才算正确。

取消细分

可以认为是相反的细分。它试图删除一次细分操作结果的边。如果在细分操作后又做了一些编辑操作，结果可能在意料之外。

迭代次数

执行取消细分操作的次数。两次迭代算是同一个细分操作，所以你通常会用偶数。

平面的

让同一个平面上的几何线消失。

角度限制

让构成角度低于该设置值的几何线消失。

所有边界

启用后，所有沿着边界面的顶点会消失掉。这样可以在用高的 角度限制 值时可以得到更好的结果。

定界

阻止某些地方的几何线消失。

法线

不要让边界区域面的法线翻转了的边消失掉。

材质

对于分配了不同材质的边界不作消失处理。

边沿

对标记为边沿的棱不作消失处理。

面数

该字段显示使用了精简修改器后，剩余面的数量。
