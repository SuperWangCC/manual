晶格修改器–knight翻译¶

The Lattice modifier deforms the base object according to the shape of a Lattice object. Objects to be deformed can be meshes, curves, surfaces, text, lattices and even particles.

小技巧

通过全选，晶格修改器可快速添加到所选物体列表中，接着最后选择晶格物体，按下:kbd:Ctrl-P，选择*晶格变形*。这样就添加了晶格修改器到所选物体上，并以晶格为父节点。

选项¶

晶格修改器。

物体
The Lattice object with which to deform the base object.
顶点组

顶点组，可选，用来限制修改器效果作用于部分原网格上。

强度

用来控制原顶点位置和变形后定点位置的混合因子。

提示¶

为什么要用晶格对一个网格变形而不是在编辑模式下变形呢？有以下几点原因：

  • 如果你的物体有大量的顶点，将很难在编辑模式下快速地编辑部分顶点。用晶格可以高效地变形大量的定点。

  • 用晶格修改器得到的平滑变形可能很难在编辑模式下手工实现。

  • 多个物体可以用同一个晶格，这样的话，你可以同时编辑多个物体。

  • 跟所有的修改器一样，它没有破坏性，也就是说对原始几何形体的作了很多改变后，你仍然可以取消改变再编辑而不会受变形的影响。

  • 晶格不会影响网格表面的纹理坐标。

注解

When using a lattice to deform particles, always remember to place the Lattice Modifier after the Particle System Modifier. Read more about the importance of the modifier stack Here.
