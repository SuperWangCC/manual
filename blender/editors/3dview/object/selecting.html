选择–幻星孤独翻译¶

Selection determines which elements will be the target of our actions. Blender has advanced selection methods. Both in Object Mode and in Edit Mode.

选择主控物体¶

Blender区分它们两者不同的选择状态:

Unselected object in black, selected object in orange, and active object in yellow.

  • In Object Mode the last (de)selected item is called the “Active Object” and is outlined in yellow (the others are orange). There is exactly one active object at any time (even when nothing is selected).

    一些动作在blender里使用活动物体时作为一个参考(例如连接操作).如果你已经有了一个选择,并且需要做一个不同的活动物体,那就使用快捷键shift-RMB来简单的重新选择它.

  • 所有其它选定的物体都是刚选好的,你可以选择任意数量的物体.

点选¶

在物体上点击RMB可以选择这个物体。

如果要同时选中多个物体，先按住SHIFT键然后再用鼠标右键分别选择即可。

在视图中,如果这个物体重叠了,你使用Alt-RMB反复通过可能的选择.

如果你想增加一个选择,它的快捷键是Shitf-Alt-RMB.

要激活一个已选定的物体,请单击Shift-RMB.

To deselect an active object, click Shift-RMB one time and hence, two clicks if the object is not active. Note that this only works if there are no other objects under the mouse. Otherwise it just adds those to the selection. There appears to be no workaround for this bug.

Border Select¶

参考

Mode: Object Mode and Edit Mode
Menu: Select ‣ Border Select
快捷键: B

BKEY激活了边界选择。使用边界选择可以选择一组通过按住LMB画出的方框里的物体。你将选中所有在方框里或者方框边界上的物体。

取消选定的物体,用MMB或者按B键之后按住Shift键再进行一次选择

取消选择使用RMB键.

例子¶

Border selecting in three steps.

在第一张图片里,边框选择已经激活了,并且已表明出显示一个虚线的十字光标.在第二张图片里,正通过与LMB绘制一个矩形的选择.这个矩形只覆盖两个立方体.最后,在第三张图中,释放LMB后选择完成.

请注意第三张图片,选定立方体的左大部分明亮的颜色.这意思就是说它是”活动物体”,在使用边框选择工具之前最后选定的物体.

提示

边框选择要添加到之前的选择,所以要合理选择矩形的内容,首先用A清除所有选项.

套索选择¶

参考

Mode: Object Mode and Edit Mode
菜单： 无菜单
快捷键: Ctrl-LMB

Lasso select is used by drawing a dotted line around the pivot point of the objects, in Object Mode.

用法¶

同时按住Ctrl键,你仅仅需要圈选你想要用LMB圈选的每个物体的顶点.

套索选择增加之前的选择.对于取消选择,用 Ctrl-Shift-LMB.

Lasso selection example.

刷选¶

参考

Mode: Object Mode and Edit Mode
Menu: Select ‣ Circle Select
热键:C（译注：我的电脑是ALT+C，C是圈选）

Circle Select is used by moving with dotted circle through objects with LMB. You can select any object by touching of circle area. It is possible to dynamically change the diameter of circle by scrolling MMB as seen in pictures below. Deselection is under the same principle - MMB. To cancel the selection use RMB or key Esc.

Circle selection.

...with huge circle.

菜单选择¶

The selection methods described above are the most common. There are also many more options accessible through the Select menu of the 3D View.

每一个都更适合某些操作.

按组选择¶

参考

Mode: Object Mode
Menu: Select ‣ Grouped
快捷键: Shift-G

Select Grouped menu.

There are two ways to organize the objects in relation to one another. The first one is parenting, and the second is simple grouping. These relationships to an artist’s advantage by selecting members of respective families or groups. Select Grouped uses the active object as a basis to select all others.

选项¶

子级

递归选择所有物体的子物体.

直接子级

选择所有物体的直接子级.

父级

如果它有一个物体，则选择该物体的父物体.

平级

选择具有与活动物体相同的父级的物体,这也可以用来选择所有的根级物体(物体没有父级).

类型（Type）

选择具有与活动物体同一类型的物体.

层

至少有一个共用层的物体.

组（Group）

如果在活动物体的一个组中，该组的一组（呈现绿色的默认主题）将被选中。

挂钩

属于主控物体的每一个挂钩.

通道
Select objects assigned to the same render pass.
颜色
Select objects with same Object Color.
属性
Select objects with same Game Engine Properties.
插帧集
Select objects included in the active Keying Set.
灯光类型

选择匹配的灯光类型。

通道索引

选择符合的物体通道索引

选择相连元素¶

参考

Mode: Object Mode
Menu: Select ‣ Linked
快捷键: Shift-L

Selects all objects which share a common data-block with the active object. Select linked uses the active object as a basis to select all others.

选项¶

物体数据

选择每一个物体,是链接同一物体数据,例如,数据块指定类型(网格,曲线等)和建立(顶点的构成基础,顶点,它们的空间位置)的物体.

材质模式

选择每一物体都是连接在同一材质的数据块.

纹理着色

选择每一物体都是连接在同一纹理的数据块.

组副本
Selects all objects that use the same Group for duplication.
粒子系统
Selects all objects that use the same Particle System
库（Library）
Selects all objects that are in the same Library
Library (Object Data)
Selects all objects that are in the same Library and limited to object data.

按类型全选¶

参考

Mode: Object Mode
Menu: Select ‣ Select All by Type
热键:没有

有了这个工具,就有可能选择某一类型的每一个可见物体.

选项¶

该类型是网格，曲线，面，融球，文本，骨架，晶格，空物体，摄像机，灯光，扬声器。

参考

All by Layer selection menu.

这个选择允许选择每一个单独的物体,在一个单一的命令里属于给定的一个层,可见或者不可见.

匹配类型选择

启用将物体添加到当前选择而不是替换当前选择.

物体所在的层。

小技巧

物体的选择

Rather than using the Select All by Layer option, it might be more efficient to make the needed layers visible and use A on them. This method also allows objects to be deselected.

其他菜单选项¶

菜单的每一级可用选项:

Select Pattern

选择名字和模式匹配的所有物体。支持通配符: * 匹配一切，?匹配任何单个的字符， [abc] 匹配字符在”abc” 字串中， [! abc] 匹配任何字符不在”abc” 中。可以选择匹配区分大小写。作为一个例子 房子 匹配任何包含 “房子” 的物体，而 地板* 匹配任何名称开始为 “地板” 的名称。

选择摄像机

选择激活的摄像机

Mirror Shift-Ctrl-M
Select the Mirror objects of the selected object eg. L.sword ‣ R.sword.
随机
Randomly selects unselected objects based on percentage probability on currently active layers. On selecting the command a numerical selection box becomes available in the Tool Shelf. It is important to note that the percentage represents the likelihood of an unselected object being selected and not the percentage amount of objects that will be selected.
Inverse Ctrl-I

选择所有物体,并同时取消已被选择的那些物体.

(De)select All A

如果任何物体已经选择了,它第一项就会被取消.否则在选择和取消选择所有可见物体之间进行切换.
