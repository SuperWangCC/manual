着色–NGENNGT 翻译¶

着色模式¶

The Viewport Shading menu.

着色指的是物体在3D视图中绘制和照明的方式

渲染

使用选定的“渲染引擎”和场景灯光照明的精确呈现。

材质
A fast approximation of the applied material.
纹理
Shows meshes with an image applied using the mesh’s active UV Map. For Cycles materials, the image is the last one selected in the Node Editor. For other render engine’s, the UV Map’s applied face texture will be shown.
实体

默认着色模式，使用实体颜色面和简单照明。

线框

使用线条组成的线框显示物体，这些线条表示面的边缘。

边界框
Only shows rectangular boxes that outline an object’s size and shape.

Except for Rendered, these shading modes are not dependent on light sources in the scene. Instead they use a simple default lighting adjusted by the Solid OpenGL Lights controls on the System tab of the User Preferences editor.

The viewport shading controls the appearance of all objects in a scene, but this can be overridden for individual objects using the Display panel in their Object Properties.

键盘快捷键

在 线框 和 实体 模式之间切换

 Z
Switches between the current and Rendered draw modes. Shift-Z

在 实体 和 纹理 模式之间切换

 Alt-Z

着色面板¶

3D View Shading Panel.

The shading panel in the Properties Region provides additional control over the way objects in the 3D View appear.

实色纹理
Display assigned face textures in the Solid shading mode. (Not available in the Cycles Render Engine).
Matcap

选择预设的着色效果，在不事先创建复杂的材质得情况下，可用在编辑和雕刻模式下的模型显示。

背面剔除

仅显示面的外侧，用来查找法线方向错误的面，尤其是在输出模型到使用单面着色的程序中的时候。

景深
Simulates a camera’s focal blur effect in the 3D View. This is only visible in a camera view. Control the effect using these options in the Properties Tab of the active camera: Focal Length, Sensor Size, Focus Object or Focus Distance, and Viewport F-stop.
环境光遮蔽

Improves the realism of the viewport image by simulating the darkening effect that occurs in crevices and corners. Typically such effects are rendered at higher quality, but this is a quick real-time preview which can help when modeling or sculpting.

使用以下设置控制环境光遮蔽效果

强度

数字越大，角落越暗。

距离

AO效果范围与角落之间的距离

衰减量

AO效果对距离的衰减强度，

采样

用于显示效果的采样值。采样值越低，显示效果越模糊，不过实际使用的采样值会进行平方，所以谨慎使用高采样值。

颜色

AO效果的颜色，调整该值可以产生从环境照明到污垢/铁锈的不同效果。
