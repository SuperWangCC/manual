重新网面化修改器–knight翻译¶

重新网面化修改器是用来产生新的网面拓扑的工具。依据输入的表面曲率作输出，但其拓扑只包含四边形。

选项¶

模式

重新网面化修改器有三种基本模式：积木、光滑和尖锐。

该示例显示了在重新网面化各个模式下的圆锥体。从左至右依次为：原始原锥体，积木模式，光滑模式，尖锐模式

输出拓扑与三种模式几乎相同；变化的是光滑效果。

积木

没作平滑处理

平滑

输出一个平滑表面。

尖锐

类似 光滑 ，但保留了尖锐的棱和角。在上述图中，圆锥体的圆形底部和锥体的顶点在尖锐模式下重建得更精确。（怀疑是尖锐acutely而不是accurately ）

八叉树深度

八叉树深度设置输出的分辨率。低值将产生相对于输入而言较大的表面，较高的值将产生一个更密集的输出。

Input mesh, and the low to high resolution output meshes.

缩放

结果还可以通过设置比例作进一步地调整；较低的值有效地降低输出分辨率。

尖锐度
Shown when using the Sharp Mode - Higher values produce edges more similar to the input, while lower values filter out noise.
平滑着色

输出面用光滑着色，而不是平坦着色。输入面的光滑/平坦着色不被保留。

删除断开连接的部分

过滤掉输出中的小的断开连接的部分。

阈值

用该选项控制断开连接的部分达到多小后必须删除。

输入网格（左）有很多噪音，所以重新网面化修改器（中心）的初始输出包含小的断开部分。启用删除断开连接部分（右）来删除那些面。

用法¶

In the modifier panel, add a Remesh modifier. The input mesh should have some thickness to it; if the input is completely flat, add a solidify modifier above the remesh modifier.
