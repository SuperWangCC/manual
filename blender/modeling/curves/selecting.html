Selecting¶

Select Menu.

Curve selection in Edit Mode is much less complex than with meshes! Mainly this is because you have only one selectable element type, the control points (no select mode needed here...). These points are a bit more complex than simple vertices, however, especially for Béziers, as there is the central vertex, and its two handles...

其基本操作与 网格 物体相同，你可以点击RMB键选择一个单一的控制点，按SHIFT+RMB追加控制点。B键框选，等等。

关于贝塞尔曲线需要注意的一点：当你选择主控制点时，两侧控制柄也会自动处于选定状态，如果不打算在曲线上做出一个角度的话，你可以把它做为一个整体来控制它。但是，如果你只选择了一个控制柄，那只有控制柄顶点被选中，允许你修改它的控制方向。

L(或ctrl - L)会扩选附近的控制点和自身的其它控制点，也就是，会把属于同一条曲线的所有控制点都选定，对于贝塞尔曲线来说，当选定控制柄时,按L键会选定曲线上所有的控制点。

选择菜单¶

With curves, all “advanced” selection options are regrouped in the Select menu of the 3D Views header. Let us detail them:

  • Random...
  • Inverse
  • Select/Deselect All
框选
All these options have the same meaning and behavior as in Object Mode (and the specifics of Border Select in Edit Mode have already been discussed here).

间隔式弃选¶

参考

Mode: Edit Mode
菜单:选择‣间隔式弃选
热键:没有

This only works if you already have at least one control point selected. Using the current selection, it will add to it every nth control point, before and after the initial selection. The “selection step” is specified in the N pop-up number button shown during the tool start.

选中/弃选首点/末点¶

参考

Mode: Edit Mode
菜单:选择‣选中/弃选首点，选择‣选中/弃选末点
热键:没有

这些命令用于切换选择曲线的首端和末端控制点，这个命令主要用于快速寻找曲线的始端（例如在曲线用作路径时）

选择下一项 /上一项¶

参考

Mode: Edit Mode
菜单:选择‣选择下一项,选择‣选择上一项
热键:没有

这个命令将会根据你当前选择的控制点追溯扩选上一个或下一个控制点（沿着曲线）。

Select More/Less¶

参考

Mode: Edit Mode
菜单:选择‣扩展选择/缩减选择
Hotkey: Ctrl-NumpadPlus/Ctrl-NumpadMinus

目的在于，基于当前所选的控制点，其目的在于缩小或扩大大这种选择。

扩展选择
for each selected control point, select all its linked points (i.e. one or two...).
缩减选择
for each selected control point, if all points linked to this point are selected, keep this one selected. Otherwise, de-select it.

这意味着两点:

  • First, when all control points of a curve are selected, nothing will happen (as for Less, all linked points are always selected, and of course, More cannot add any). Conversely, the same goes when no control points are selected.

  • 第二，这两个命令不会“跳出”曲线，（就算是扩选，它们也不会超出自己的曲线范围跑到另一条曲线上）。
